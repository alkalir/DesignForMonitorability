# Papers to be Classified with MONICA Tool Parameters

In this README you can find the current list of papers that are scheduled to be classified with MONICA tool parameters in the next future. Feel free to contribute!

You should:
1. Download the paper
2. Read it
3. Assign the MONICA parameters (see the website [https://monicatool.cloud/](https://monicatool.cloud/) and the description of each parameter in the folder [MONICATool/parametersDescription](https://github.com/alkalir/DesignForMonitorability/tree/main/MONICATool/parametersDescription) of this repo)
4. Submit it — please send to [giacomo.valente@univaq.it](mailto:giacomo.valente@univaq.it) *(pull requests are not accepted for now)*
