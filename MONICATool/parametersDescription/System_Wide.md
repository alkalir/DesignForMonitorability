# System-Wide

The **System-Wide** parameter refers to the capability of the monitoring system to monitor a heterogeneous environment composed of General-purpose Processors (GPPs) and Single-purpose Processors (SPPs).

This parameter can be set with one value from the list below.

## Possible Values

- **YES:** the monitoring system is capable of monitoring a heterogeneous environment composed of GPPs and SPPs.
- **NO:** the monitoring system is not capable of monitoring a heterogeneous environment composed of GPPs and SPPs.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
