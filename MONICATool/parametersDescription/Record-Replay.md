# Record-Replay

The **Record-Replay** parameter refers to whether the on-chip monitoring system is able to record and replay the behaviour of the target under analysis.

This parameter can be set with one value from the list below.

## Possible Values

- **YES:** the monitoring system is capable of recording and replaying the behaviour of the target under analysis.
- **NO:** the monitoring system is not capable of recording and replaying the behaviour of the target under analysis.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
