# Availability

The **Availability** parameter refers to whether the monitoring system implementation is available and in which form.

This parameter can be set with one or more values from the list below.

## Possible Values

- **Closed Source:** the monitoring system implementation is available but not publicly accessible as source code.
- **Open Source:** the monitoring system implementation is publicly available as source code.
- **NO:** the monitoring system implementation is not available.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
