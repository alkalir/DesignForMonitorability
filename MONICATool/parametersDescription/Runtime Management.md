# Runtime Management

The **Runtime Management** parameter refers to whether the monitoring system offers also a runtime management part or not. In case of runtime management actions, also the type of it is required. This action can be considered as a control action, based on the observed behaviors.

This parameter can be set with one or more values from the list below.

## Possible Values

- **Temperature Control:** the monitoring system offers a runtime management part aimed at controlling the temperature of the system.
- **Correct Execution:** the monitoring system offers a runtime management part aimed at ensuring the correct execution of the monitored application.
- **Reliability:** the monitoring system offers a runtime management part aimed at improving the reliability of the system.
- **NO:** the monitoring system does not offer a runtime management part.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
