# Parameters Description

This folder contains the descriptions of the parameters used to classify on-chip monitoring systems (OCMSs) with the [MONICA tool](https://monicatool.cloud/).

Each parameter is documented in a dedicated Markdown file. For each parameter, the description includes the definition, the set of possible values, and, where applicable, examples.

To classify an OCMS, read the description of each parameter and assign the appropriate value(s) according to the characteristics of the monitoring system under analysis.
