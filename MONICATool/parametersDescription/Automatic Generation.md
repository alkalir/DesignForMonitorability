# Automatic Generation of Monitors

The **Automatic Generation** parameter refers to whether the monitoring system can be automatically generated by a framework or tool (e.g., via high-level synthesis, automatic hardware synthesis, or automatic software generation flows), without requiring manual implementation by the designer.

This parameter can be set with one value from the list below.

## Possible Values

- **YES:** the monitoring system can be automatically generated.
- **NO:** the monitoring system cannot be automatically generated.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
