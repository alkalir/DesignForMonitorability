# Design Cost

The **Design Cost** parameter refers to the effort needed to design and implement the monitoring system. It strongly depends on the flexibility of the selected solution.

This parameter can be set with one value from the list below.

## Possible Values

- **Low:** the monitoring system is automatically generated or requires only configuration of an existing framework.
- **Medium:** the monitoring system requires partial modifications to an existing framework or template.
- **High:** the monitoring system requires design and implementation from scratch.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
