# Extensibility

The **Extensibility** parameter refers to the capability of the monitoring system to be extended with new monitoring functionalities without requiring a complete redesign.

This parameter can be set with one value from the list below.

## Possible Values

- **YES:** the monitoring system can be extended with new monitoring functionalities.
- **NO:** the monitoring system cannot be extended with new monitoring functionalities.
- **Not Applicable:** This value should never be set.
- **Not Declared:** this parameter is not declared for the considered monitoring requirements, i.e., no value has been specified for it. This value must be set in case the other ones are not set.
